> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 DAG 节点的"参数查看"和"总结查看"从内联展开改为居中模态弹窗。

**Architecture:**
- 新建 `NodeDetailModal.tsx` 组件，在 DAGCanvas 级别渲染，完全解耦 DAGNode
- DAGNode 只保留按钮，移除内联展开逻辑，不再需要 `expandedWidths` 追踪
- 弹窗状态在 `DAGCanvas.tsx` 中管理，支持切换节点内容而不关闭弹窗

**Tech Stack:** React, TypeScript, CSS-in-JS inline styles, ReactMarkdown + remarkGfm

```typescript
// ── Focus Trap ────────────────────────────────────────────────────────────────
function useFocusTrap(ref: React.RefObject<HTMLElement | null>, active: boolean) {
  useEffect(() => {
    if (!active || !ref.current) return;
    const el = ref.current;
    const focusable = el.querySelectorAll<HTMLElement>(
      'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
    );
    const first = focusable[0];
    const last = focusable[focusable.length - 1];
    first?.focus();

    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key !== 'Tab') return;
      if (e.shiftKey) {
        if (document.activeElement === first) { e.preventDefault(); last?.focus(); }
      } else {
        if (document.activeElement === last) { e.preventDefault(); first?.focus(); }
      }
    };
    el.addEventListener('keydown', handleKeyDown);
    return () => el.removeEventListener('keydown', handleKeyDown);
  }, [active, ref]);
}
```

```typescript
  const parentId = (n.data as DAGNode).parentId ?? 'main-agent';
  const parentIdx = level1.findIndex(q => q.id === parentId);
  const parentX = parentIdx >= 0
    ? centerX + (parentIdx - (level1.length - 1) / 2) * 280
    : centerX;

  const siblings = level2.filter(t => (t.data as DAGNode).parentId === parentId);
  const toolIdxInQuery = siblings.indexOf(n);
  const xStep = 280;
  const offsetX = (toolIdxInQuery - (siblings.length - 1) / 2) * xStep;
  result.push({ ...n, position: { x: parentX + offsetX, y: 20 + yStep * 2 } });
}
```
