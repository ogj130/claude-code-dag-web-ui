工具执行结果展示方案

Claude Code 工具调用后的输出怎么呈现给用户？

A

xterm.js 终端模拟器

完全模拟 CLI 终端界面，复刻 Claude Code 命令行体验，工具输出原样打印

  • ✅ 零学习成本，完全复刻 CLI 体验
  • ✅ Claude Code 输出格式无需适配
  • ⚠️ 无法结构化解析工具结果
  • ⚠️ 失去了"工具卡片"的结构化展示优势
B

结构化工具卡片 + 富文本

每个工具调用渲染为一张卡片：工具名、参数、执行状态、结果摘要，可展开查看详情

  • ✅ 信息结构化，一眼看清每个工具干了什么
  • ✅ 可折叠/展开，支持大量工具调用
  • ⚠️ 需要解析 Claude Code 输出格式
  • ⚠️ 失去终端原生的"流式打字机"感
C

混合模式（推荐）

左侧 DAG 实时进度视图 + 右侧可切换：终端视图 / 工具卡片视图，两种模式自由切换

  • ✅ 兼得两种体验，按需切换
  • ✅ 概览用卡片，详细用终端
  • ⚠️ UI 复杂度稍高
